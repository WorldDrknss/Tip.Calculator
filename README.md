# Steps to clone this project

1. Clone Repository
    ```sh
    git clone https://lab.skynetinc.tech/WorldDrknss/Tip.Calculator.git
    ```

2. `cd` into the cloned repository
    ```sh
    cd Tip.Calculator
    ``` 

3. Launch Tip Calculator
    ```sh
    python3 main.py
    ```
